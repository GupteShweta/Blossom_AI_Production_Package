## 🚀 Overview
Blossom AI is a production-grade ecological intelligence platform integrating:
- Firebase Authentication & Firestore
- TensorFlow Lite plant/animal recognition
- Wikipedia REST API integration
- NASA & NOAA climate data APIs
- Android Material 3 UI
- Bootstrap 5 Web Interface

## 🧠 Features
- AI biodiversity recognition
- Climate data correlation
- Ecological observation repository
- User-based research contribution system

## 🌍 Climate APIs Integrated
- NASA EarthData API
- NOAA Climate Data Online API

## 🏗 Architecture
Android/Web → Firebase Auth → Firestore → Storage → ML → Climate APIs → Wikipedia API
